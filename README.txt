========================================
  Textube - YouTube Subtitle Extractor
  v2.1
========================================

YouTube動画から字幕を抽出してコピーできるChrome拡張機能です。

v2.1の新機能:
- URL入力による字幕取得（YouTubeページに居なくてもOK）
- 要約プロンプトをUIから編集・保存（popup右下の⚙ボタン）


--------------------
 必要なもの
--------------------

- Python 3.12以上
  https://www.python.org/downloads/
  ※インストール時に「Add Python to PATH」にチェック

- Google Chrome


--------------------
 インストール手順
--------------------

1. install.bat をダブルクリック
   → 必要なライブラリがインストールされます
   → サーバーがバックグラウンドで起動します

2. Chromeを開く

3. アドレスバーに入力: chrome://extensions

4. 右上の「デベロッパーモード」をON

5. 「パッケージ化されていない拡張機能を読み込む」をクリック

6. このフォルダ内の「extension」フォルダを選択

7. 完了！


--------------------
 使い方 A: YouTubeページから取得
--------------------

1. YouTubeで動画を開く

2. ブラウザ右上のTextubeアイコンをクリック

3. 「Extract Subtitles」ボタンをクリック

4. 字幕が表示されたら：
   - Copy: 字幕のみコピー
   - Copy with Prompt: 要約用プロンプト付きでコピー
   - Download: テキストファイルとして保存


--------------------
 使い方 B: URL入力で取得 (v2.1〜)
--------------------

YouTubeページにいなくても字幕を取得できます。

1. 任意のページでTextubeアイコンをクリック

2. 「URLから取得（任意）」欄にYouTube URLまたは
   11桁の動画IDを貼り付け

3. 緑のヒント「動画ID: xxxxxxxxxxx」が出ればOK

4. 「Extract Subtitles」をクリック

※ URL欄に入力がある場合は、現在のタブよりURL欄が優先されます


--------------------
 プロンプト編集 (v2.1〜)
--------------------

「Copy with Prompt」で挿入される要約プロンプトをUIから
編集できます。

1. popup右下の⚙ボタンをクリック

2. モーダルでプロンプトを編集

3. ボタン操作：
   - 保存       : 編集内容を保存（次回以降も有効）
   - 初期値に戻す : 編集欄を初期プロンプトに戻す（保存ボタンで確定）
   - キャンセル  : 変更を破棄して閉じる


--------------------
 アンインストール
--------------------

1. uninstall.bat をダブルクリック
   → 自動起動が解除されます

2. Chromeで chrome://extensions を開く

3. Textubeの「削除」をクリック

4. このフォルダを削除


--------------------
 トラブルシューティング
--------------------

Q: 「Server offline」と表示される
A: install.bat を再実行してください

Q: 字幕が取得できない
A: その動画に字幕がない可能性があります

Q: Pythonがないと言われる
A: python.org からPythonをインストールしてください
   インストール時に「Add Python to PATH」にチェック


--------------------
 注意事項
--------------------

- PCを再起動してもサーバーは自動で起動します
- 字幕のない動画では使用できません
- 一部の動画では字幕取得がブロックされる場合があります
